## **1. exa-py 라이브러리**
  ```exa.search(...)```
>  > query : 검색 내용
>  > num_results : 결과 개수 설정
>  > startPublishedDate/endPublishedDate : 해당 기간 내에 발행된 페이지
>  > include_domains/exclude_domains : 특정 도메인을 검색에 포함/제외
>  > category : 특정 유형 검색
  * AI 검색 엔진으로 api 키 필요
